## Getting Started

### Add Env Variables

Add your Twilio Account SID and Auth Token. Then add the Twilio phone number and your email.

```bash
TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=

DEMO_PHONE=
DEMO_EMAIL=
```

### Run Server

```bash
npm run dev
```
